# Sponsors

## What helps most
- actuators, sensors, mechanical components
- manufacturing cost (CNC/3D printing/materials)
- travel/competition operation cost
- GPU/server resources for training/experiments

- **Gold**: logo placement, social/media credits
- **Silver**: sponsor list placement
